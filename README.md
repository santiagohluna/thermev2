# THERMEV2

## Purpose

This program computes the thermal evolution of the Earth's interior using a 1D parametrized thermal model based on the works by [Schubert et al. (2001)](https://doi.org/10.1017/CBO9780511612879), [Driscoll &Bercovici (2014)](https://doi.org/10.1016/j.pepi.2014.08.004), and [Driscoll & Barnes (2015)](https://doi.org/10.1089/ast.2015.1325).
